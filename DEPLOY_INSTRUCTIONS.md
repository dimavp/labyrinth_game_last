# Инструкция по деплою на CapRover

## Готовый файл для деплоя
✅ Файл `labyrinth.tar` создан и готов к загрузке

## Шаги для деплоя:

### Вариант 1: Через веб-интерфейс CapRover

1. Откройте CapRover Dashboard (обычно это captain.dimavp.ru или подобный URL)
2. Войдите в систему
3. Выберите ваше приложение (например, `labyrinth-vs-code`)
4. Перейдите в раздел "Deployment"
5. В секции "Method 3: Deploy via tarball" загрузите файл `labyrinth.tar`
6. Нажмите "Upload & Deploy"
7. Дождитесь завершения деплоя

### Вариант 2: Через CLI

Если у вас установлен CapRover CLI:

```bash
# 1. Войдите в CapRover (если еще не вошли)
caprover login

# 2. Задеплойте tar-файл
caprover deploy -t ./labyrinth.tar -a labyrinth-vs-code
```

## После деплоя

Приложение будет доступно по адресу:
- https://labyrinth-vs-code.apps.dimavp.ru/

## Файлы в архиве

- index.html - главная страница с новым UI
- style.css - стили включая кнопки сложности
- script.js - логика игры с системой сложности

## Проверка после деплоя

1. Откройте игру в браузере
2. Проверьте, что видны кнопки 1-4 для выбора сложности
3. Попробуйте переключить каждый уровень
4. Убедитесь, что лабиринт меняется соответственно
