# Система сложности игры "Лабиринт для Иришки"

## Обзор изменений

Добавлена возможность выбора сложности игры от 1 до 4 непосредственно в интерфейсе игры.

## Уровни сложности

### Уровень 1 - Легкий
- Размер лабиринта: 10x10 клеток
- Размер ячейки: 50px
- Размер холста: 500x500px
- Идеально для начинающих

### Уровень 2 - Средний (по умолчанию)
- Размер лабиринта: 15x15 клеток
- Размер ячейки: 40px
- Размер холста: 600x600px
- Оптимальный баланс сложности

### Уровень 3 - Сложный
- Размер лабиринта: 20x20 клеток
- Размер ячейки: 30px
- Размер холста: 600x600px
- Для опытных игроков

### Уровень 4 - Эксперт
- Размер лабиринта: 25x25 клеток
- Размер ячейки: 24px
- Размер холста: 600x600px
- Максимальная сложность

## Визуальные особенности

- **Кнопки выбора сложности**: 4 кнопки с номерами 1-4
- **Активная кнопка**: Выделена зеленым цветом с подсветкой
- **Hover-эффект**: Синий цвет при наведении с масштабированием
- **Плавные переходы**: Все изменения анимированы

## Функциональность

1. **Динамическое изменение**: При нажатии на кнопку сложности автоматически генерируется новый лабиринт
2. **Визуальная обратная связь**: Активная кнопка всегда подсвечена
3. **Мгновенное применение**: Изменения вступают в силу сразу же
4. **Адаптивный холст**: Размер холста и клеток адаптируется под выбранную сложность

## Как использовать

1. Откройте игру в браузере
2. Выберите желаемый уровень сложности (1-4)
3. Лабиринт автоматически сгенерируется с новыми параметрами
4. Играйте и наслаждайтесь!

## Технические детали

### Файлы изменены:
- `index.html` - добавлен UI для выбора сложности
- `style.css` - добавлены стили для кнопок сложности
- `script.js` - реализована логика динамического изменения параметров игры

### Ключевые изменения в коде:
- Добавлен объект `difficultySettings` с параметрами для каждого уровня
- Переменные `COLS`, `ROWS`, `CELL_SIZE` теперь динамические
- Функция `setup()` применяет настройки выбранной сложности
- Добавлены обработчики событий для кнопок сложности

## Тестирование

Локальный сервер запущен на: http://localhost:8000

Для тестирования:
1. Откройте http://localhost:8000 в браузере
2. Проверьте все уровни сложности
3. Убедитесь, что игра корректно работает на всех уровнях
